Website HMPSI Laravel 8
Requirement :
1. PHP 7.4.3
2. Xampp/Laragon
3. Composer

Cara Menjalankan :

1. Clone Project dari github
2. Buat Branch baru
3. Buka terminal 
4. ketikkan "composer install"
5. ketik "composer update"
6. ketik "php artisan key:generate"
7. ketik "php artisan serve"
